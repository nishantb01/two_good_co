# two_good_co
Two Good Co Landing Page Clone

This project is a front-end recreation of the Two Good Co website landing page, built using HTML, CSS, and JavaScript. The purpose of this project was to practice modern website layouts, animations, and responsive design by replicating a real-world website.

🚀 Features

Modern landing page layout

Smooth UI interactions and animations

Clean and structured HTML/CSS components

Responsive design for multiple screen sizes

Interactive elements using JavaScript

🛠️ Tech Stack

HTML5

CSS3

JavaScript

📚 Learning Outcomes

Understanding modern website structure

Implementing interactive UI elements

Improving CSS styling and layout techniques

Practicing real-world website replication

📂 Installation

Clone the repository

git clone https://github.com/nishantb01/two_good_co.git

Navigate to the project folder

cd two_good_co

Open index.html in your browser.
